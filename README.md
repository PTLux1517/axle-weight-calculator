# Axle Weight Calculator

A job site specific tool for calculating load weight
distribution, tailored to the needs of such anonymous company, but also available
open source under the following AGPL license for any who may also find it useful.

This has been developed to be used client side only, in order to enable downloading
and running on any non-networked device with a browser.

[web app here](https://ptlux1517.github.io/axle-weight-calculator/)
